# Vector Store Production Options

| Store | Best For | Cost | Type |
| :--- | :--- | :--- | :--- |
| **Chroma** | Local dev, prototyping | Free | Local |
| **FAISS** | Speed, in-memory | Free | Local |
| **Pinecone** | Production, managed | $70+/mo | Cloud |
| **Weaviate** | Self-hosted, hybrid | Free/Paid | Hybrid |
| **Qdrant** | Performance, filtering | Free/Paid | Hybrid |

> **Recommendation:** Start with Chroma. Move to Pinecone/Qdrant for production scale.

# Chunk Size Guidelines

| Use Case | Chunk Size | Overlap | Why |
| :--- | :--- | :--- | :--- |
| **Q&A / FAQ** | 500 - 800 | 50 - 100 | Precision |
| **Documentation** | 1000 - 1500 | 200 | Context |
| **Code** | 1500 - 2000 | 200 | Functions |
| **Books / Articles** | 800 - 1200 | 150 | Balance |

### Chunk Size Scale

- **Too Small**
- **Optimal**
- **Too Large**
